# [実験01（知識無しのSAP-netTest）](02_paper_01.md)

この実験は、選択的転移強化学習において転移先の強化学習済み知識が存在しない状態で行われる実験である。

目的は、転移先が用意された場合において、どのように転移が可能かを検討し、基盤となるSAP-netの実現可能性を判断することだ。
